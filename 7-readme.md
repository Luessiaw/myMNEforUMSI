为了规范，这里只实现有限的功能：

- 源
  - 单个点状的源
  - 单个时刻
  - 方向为切向
- 介质
  - 三维单层均匀球体
  - 划分格点
  - 不使用 BEM 方法而直接计算前向模型
- 探头
  - 无体积的 OPM, 区分零磁和地磁
  - 只考虑白噪声
  - 均匀分布
  - 不使用差分探测
  - 不考虑磁场方向的梯度效应
- 地磁场环境
  - 外磁场不含时
  - 可存在一阶梯度，但是未知
  - 磁场方向已知，不考虑其误差


根据以上，主要参数：
- 源的分布
  - 强度
  - 位置
- 探头
  - 灵敏度
  - 带宽
  - 数量
- 介质
  - 格点边长
- 磁场环境
  - 地磁场大小
    - 地磁场大小误差
  - 地磁场方向
    - 地磁场方向误差
  - 地磁场梯度
  - 零磁环境：地磁场大小、梯度均为零，方向为径向
- 模拟设置
  - 模拟次数
  - 源分布的范围
    - 深度
    - 角度（轴对称，一个自由度）

做图方式另外考虑。
